
Requirement
====

* Homebrew
  * http://brew.sh/
* Cask for Emacs.
  * http://cask.readthedocs.org/
  * Cask requires Emacs 24+.
    * `$ brew install emacs`

Setup
====

* `$ cd .dotfiles/`
* `$ /bin/sh setup.sh`
