
Requirement
====

* Homebrew
  * http://brew.sh/
* Cask for Emacs.
  * http://cask.readthedocs.org/
  * Cask requires Emacs 24+.
    * `$ brew install emacs`
  * `$ cask install`
* EditorConfig
  * `$ brew install editorconfig`
* bash-completion
  * `$ brew install bash-completion`
* maltimarkdonw
  * `$ brew install multimarkdown`
